# Brain v0.4 Release Track

This document tracks the v0.4 release milestones.

## Milestone 1: Extraction Quality ✅
- Improve synthesis template normalization
- Enhance fallback formatters for all template types
- Ensure consistent markdown output structure

## Milestone 2: Behavior-Level Regression Coverage (In Progress)
- Expand smoke test coverage for edge cases
- Add integration tests for key workflows
- Test error paths and API failures
- Verify cache invalidation behavior

## Milestone 3: Large-Vault Responsiveness ✅
- Implement count caching with mtime-based invalidation
- Optimize context collection for vault-wide operations
- Add file filtering to exclude Brain-generated content from recursion

## Milestone 4: UX Consistency ✅
- Standardize error handling patterns
- Unify modal and sidebar interactions
- Ensure consistent status reporting

## Future Considerations
- Keyboard shortcuts for common sidebar actions
- Collapsible sidebar sections
- Loading states for async operations
- Settings validation and feedback
- Custom synthesis templates
- Batch inbox operations
